À propos

DevOps Analyzer est un outil développé dans le cadre d'un projet scolaire pour analyser automatiquement des projets. Son objectif ? Vérifier les bonnes pratiques en testant :

  • • La présence et le fonctionnement des pipelines CI/CD
  • • La qualité du code via les tests unitaires
  • • Les configurations Docker
  • • La maintenabilité du projet

Spécificités :

  • ✔ Génère des rapports détaillés au format JSON
  • ✔ Rapports faciles à lire, à exploiter et à partager
  • ✔ Génère une évaluation automatique basée sur l'analyse du projet

Un outil par des étudiants, pour des étudiants – parce que l'apprentissage passe aussi par l'auto-évaluation.
